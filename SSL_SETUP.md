# SSL Setup for Toll Management System

এই গাইড আপনাকে Django অ্যাপ্লিকেশনটি SSL-এর সাথে চালানোর জন্য সাহায্য করবে।

## প্রয়োজনীয় ফাইলগুলি

আপনার সার্টিফিকেট ফাইলগুলি `cert` ফোল্ডারে রাখুন:

```
cert/
├── private.key          # আপনার প্রাইভেট কী ফাইল
├── certificate.crt      # আপনার সার্টিফিকেট ফাইল
└── combined_cert.crt    # সার্টিফিকেট + চেইন ফাইল (Sub CA + Root CA)
```

## ধাপ ১: সার্টিফিকেট প্রস্তুত করা

প্রথমে সার্টিফিকেট ফাইলগুলি প্রস্তুত করুন:

```bash
python prepare_cert.py
```

এই স্ক্রিপ্টটি আপনার সার্টিফিকেট ফাইলগুলি যাচাই করবে এবং একটি সম্মিলিত SSL সার্টিফিকেট তৈরি করবে।

## ধাপ ২: SSL সার্ভার চালানো

### পদ্ধতি ১: কাস্টম Django কমান্ড ব্যবহার করে

```bash
python manage.py runsslserver
```

### পদ্ধতি ২: SSL স্ক্রিপ্ট ব্যবহার করে

```bash
python run_ssl.py
```

### পদ্ধতি ৩: সরাসরি SSL স্ক্রিপ্ট

```bash
python run_ssl_simple.py
```

## সার্ভার অ্যাক্সেস

SSL সার্ভার চালানোর পর, আপনার অ্যাপ্লিকেশন এখানে অ্যাক্সেস করা যাবে:

```
https://toll.sdlbdcloud.com:8000
```

## গুরুত্বপূর্ণ নোটস

1. **ডোমেইন নাম**: আপনার সার্টিফিকেট `toll.sdlbdcloud.com` ডোমেইনের জন্য ইস্যু করা হয়েছে
2. **পোর্ট**: ডিফল্ট পোর্ট 8000, আপনি পরিবর্তন করতে পারেন
3. **ফায়ারওয়াল**: পোর্ট 8000 (বা আপনার নির্বাচিত পোর্ট) ফায়ারওয়ালে খোলা আছে কিনা নিশ্চিত করুন
4. **DNS**: `toll.sdlbdcloud.com` ডোমেইন নামটি আপনার সার্ভারের IP ঠিকানায় নির্দেশিত হয়েছে কিনা নিশ্চিত করুন

## সমস্যা সমাধান

### সার্টিফিকেট ফাইল খুঁজে পাওয়া যায় না
- নিশ্চিত করুন যে সব সার্টিফিকেট ফাইল `cert` ফোল্ডারে আছে
- ফাইল নামগুলি সঠিক কিনা যাচাই করুন

### SSL সংযোগ ত্রুটি
- সার্টিফিকেট ফাইলগুলি সঠিক ফরম্যাটে আছে কিনা যাচাই করুন
- প্রাইভেট কী ফাইলটি সঠিক কিনা নিশ্চিত করুন

### পোর্ট অ্যাক্সেস করা যায় না
- ফায়ারওয়াল সেটিংস যাচাই করুন
- পোর্টটি অন্য অ্যাপ্লিকেশন ব্যবহার করছে কিনা দেখুন

## প্রোডাকশন ডেপ্লয়মেন্ট

প্রোডাকশনে, নিম্নলিখিত বিষয়গুলি বিবেচনা করুন:

1. **Gunicorn/uWSGI**: Django development server এর পরিবর্তে Gunicorn বা uWSGI ব্যবহার করুন
2. **Nginx**: Reverse proxy হিসেবে Nginx ব্যবহার করুন
3. **SSL Termination**: Nginx এ SSL termination করুন
4. **Static Files**: Static ফাইলগুলি CDN বা Nginx এ serve করুন

## সহায়তা

- সার্টিফিকেট ফাইলগুলির ফরম্যাট
- এরর মেসেজ
- সার্ভার লগ
